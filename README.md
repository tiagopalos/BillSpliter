# Bill Splitter

Bill Splitter is a intuitive web application for splitting bills


# Features

##  Bill Management

Multi-Currency Support
Add Bill Items: Easily add items with custom names and prices
Undo/Redo Functionality

## People Management

Add as many people as needed to split the bill
Dynamic List: Easy-to-manage participant list with individual delete options
Clear All: Quick reset

## Summary and Breakdown

Itemized Summary: View all items with individual prices
Total Calculation
Per-Person Split: Equal division of the total bill among all participants
Visual Analytics: Interactive pie chart showing expense breakdown by item
Most Expensive Item: Highlights the most expensive item 

## Export Options
PDF and PNG Export

# Usage

## Adding Bill Items

Select your currency from the dropdown
Enter the item name (e.g., "Pizza", "Drinks")
Enter the price
Click "+ Add" to add the item or press Enter
Use Undo/Redo buttons to correct mistakes

## Managing People

Defaul People in the list
Enter a person's name in the input field
Click "+ Add Person" to add them or press "Enter"
Double click Name to Eidt 
Use the trash icon to remove individual people
Use "Clear All" to remove everyone

## Viewing Results

Summary: See all items and the total bill
Per Person: View the split amount each person owes
Expense Breakdown: Visual pie chart representation
Most Expensive: Quick view of the highest-priced item
